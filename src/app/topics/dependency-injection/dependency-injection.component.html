Depedencies are service or objects that a class needs to perfom its function. Depedency Injection(DI) is a design pattern in which a class requests depedencies from external sources rather than creating them. Its like adding a new functionality/extension to the application.

How to implement injectable service.
  1. Create a service file : log.service.ts
  2. Import Injectable from angular.core.
  3. Implement Injectable decorator with providedIn as root as we are gonna inject this service at root.
  4. Export a class and add implementation.
  5. In app.module inject this service class in providers. Where its visible in component.
  6. In app component declare the object of this class.
