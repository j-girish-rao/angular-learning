Property binding moves a value in one direction. Value goes from class property to target html. Property bidning helps set values of HTML elements or directives. Following are examples of Property binding.
Here in button [disabled]="isUnchanged", disabled is the property and isUnchanged is the value thats been passed from component.
With interpolation you can achieve the same result. button [disabled]="isUnchanged" //This is property binding.
button disabled="{{disabledText}}" //This is interpolation. Both work as same until evaluation of template expression shouldn't have no visible side effects.
