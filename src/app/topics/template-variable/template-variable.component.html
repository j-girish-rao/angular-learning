Helps use data from one part of the template to other part of template. Its a variable which is created in and identify component or element within template itself.

This can be used in case of respond to user input or finetune the applications form.

A # symbol is used to declare a template variable. Eg : {{templateVariableExample}} // Here phone is the template variable and its value can be accessesed by phone.value.



What is the scope of template variable ?
This can be used/accessed any where within its surrounding template. Structural template such as ng-If, ng-For or ng-Template acts as templat boundary. You cannot access template variables outside of these boundaries.
