Reactive forms provide model driven approach to handle form inputs whos values change over time, each change to the form state returns the new state which maintaines the integrity of the model between changes.
Example : Simple form with 1 field either input name or click button to set value. The Form value which is set for name is accessible in html and component as well.

Reactive forms provide two ways of grouping multiple related controls into a single input form.
1. FormGroup defines a fixed set of controls that can be managed together.
2. FormArray defines a dynamic form where we can add and remove controls at runtime.
