Text Interpolation
-------------------
Text interpolation is one way data binding, sending the data from typescript to the html view.
{{templateExpression}}
When do we need text interpolation ?
We need text interpolation when we have to something dynamic in out html template.
Following is the example of text interpolation where the timer is displayed from typescript.
    Example 1 : {{title}}   // {{example_1}}
    Exmaple 2 : {{1+10}}    // {{example_2}}
    Example 3 : {{"Girish" + " J"}}   // {{example_3}}
    Example 4 : {{sumOfNumbers(11,100)}}    // {{example_4}}
Note : Here we cant use assignment operators like =, +=, -=, new, typeOf, instanceOf operators, chaining operators like --, ++
