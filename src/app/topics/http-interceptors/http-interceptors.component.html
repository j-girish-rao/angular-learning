Http interceptors intercept and transform http requests from application to server. It can perform a variety of tasks like authentication, logging for every http request.
Create a class that implements 'HttpInterceptor' and implements intercept function. This function accepts two parameters request and next, where request is current ongoing request and nect to handle the request.
In app module providers handle this interceptor when you need to provide what all requests it needs to handle.
