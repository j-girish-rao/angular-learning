Directives are classes that add additional behaviour to elements in angular application. With angular application build in directives we can manage forms, lists, styles what user can see.

The different type is angular directives are as follows :
1. Components directives : With a template. This type of dirrective is the most common directive type.
2. Attribute directives : Directives that change appearence or behaviour of an element, component or an another directive.
3. Structural directives : Directives that change DOM layout by adding and removing elements in a html page.
