Animation provide the illumination of motion. Elements change styling overtime. Angular can animate any property that the browser considers animatable. This includes porition, sizes, transforms, colors, borders etc.

To implement animations import browser animation module in app module. And import required in component.
To implement animation define animate in component which trigerres animation. In this case 'openClose'. Following is an example.
{{isOpen}}
