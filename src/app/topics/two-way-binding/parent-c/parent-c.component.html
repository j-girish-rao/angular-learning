Two way binding helps in sending data from component to html and vice versa. With this data or html can be taken as input and sent back to component.
NOTE : Two way binding is short hand of property binding and event binding.
Systex of achieving two way binding is [(variable_name_goes_here)]. Which is dataMember[] and memberFunction().
