Pipes are functions that accepts a input value and return a formatted value. Pipe has to implement Pipe decorator. Where it has to implement transform method. This is where the changes to input are applied.
Syntax : {{pipeSyntax}} 
    For example : 
    a. Currency 
        Before : 100 | After : $100 // Dollar as prefix.
    b. Weight
        Before : 38.1903 | After 38.2 Kg // Round of decimal to single digit and postfix Kg to indicate weight. 
                                            Likewish it can applied to format temprature and various other fields.

    Following are few builtIn pipes.
        a. DataPipe
        b. UpperCasePipe
        c. CurrencyPipe
        d. PercentPipe

    Use this command to generate pipe.
        ng g p greeting-pipe

Pipes are executed only after it detects pure change to the input value.
