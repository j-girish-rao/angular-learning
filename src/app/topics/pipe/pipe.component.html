Pipes
-----
Pipes are functions that accepts a input value and return a formatted value. 
    For example : 
    a. Currency 
        Before : 100 | After : $100 // Dollar as prefix.
    b. Weight
        Before : 38.1903 | After 38.2 Kg  // Round of decimal to single digit and postfix Kg to indicate weight.
    Likewish it can applied to format temprature and various other fields.
BuiltIn pipes from Angular
Date before pipe format : {{beforeFormatDate}}
Date after pipe format : {{beforeFormatDate | date}} // Date format | date
Date after pipe format : {{beforeFormatDate | date : 'short'}} // Date format | date : 'short'
Date after pipe format : {{beforeFormatDate | date : 'long'}} // Date format | date : 'long'

Pipes are executed only after it detects pure change to the input value.

Custom Pipes
Here in example weight is the pipe that is created to postfix weight and round off value to single decimal place.
Before : {{weight_stu1}} | After : {{weight_stu1 | weightPipe}}
Before : {{weight_stu2}} | After : {{weight_stu2 | weightPipe}}
