Directives are classes that add additional behaviour to elements(how to behave and what to do) in angular application. With angular application build in directives we can manage forms, lists, styles what user can see.

The different type is angular directives are as follows :

1. Components directives
A special type of directive that defines a reusable UI element with its own template, logic, and styling, forming the building blocks of an Angular application.
Eg : Component

2. Attribute directives
Directives that change appearence or behaviour of an element, component or an another directive.
Eg : NgClass, NgStyle, NgModel.

3. Structural directives
Directives that change DOM layout by adding and removing elements in a html page.
Eg : NgIf, NgFor, NgSwitch.
