Are responsible for html layout. They shape or reshape the DOM structure by adding, removing and manipulating the host elements.
1. Built-in structural directives

1. NgIf : Conditionaly created or disposes the sub-elements from the template. When its NgIf is false angular removes an element and its descendants from the DOM. Angular then disposes of their components which frees up memory and resources.
Syntax : {{ngIfSyntax}}
Example : {{isShowText}} => Show span data. // As isShowText is true the span is visible. Adds DOM element to parent DOM.
Example : {{isHideText}} => Hide span data. // Nothing is visible because isHideText is false so this DOM element is not included.

2. NgFor : Repeat a node for each item in a list.
Syntax : {{ngForSyntax}}
Example : {{student.name}}
Syntax : {{ngOtherForSyntax}}
Example : {{i+1}}.{{student.name}}

3. NgSwitch : A set of directive that switch among alternative views. Angular puts only the selected element into the DOM. ngSwitch is a set of three directives ngSwitch(an attrivute directive that changes the behaviour of its companion directive), ngSwitchCase (structural directive that adds its element to DOM when its bound value is equal to switch value and removed its bound value when it doesnt equal the switch value) and ngSwitchDefault (that adds its element to the DOM when there is no selected switch case).
Syntax : 
{{ngSwitchSyntax}}
